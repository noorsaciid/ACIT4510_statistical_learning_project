# ACIT4510_statistical_learning_project
The project analyzes the Mental Health and Lifestyle Habits dataset using regression, classification, and PCA to identify factors influencing stress and well-being. It applies ridge and lasso regularization with cross-validation in Python, following ACIT4510 and ISLP methods to build accurate, interpretable, and data-driven models.

# Technologies Used
- **Python**
- **Pandas** and **NumPy**
- **Scikit-Learn**
- **Seaborn** and **Matplotlib**
- **Jupyter Notebook**
- **Power BI**
